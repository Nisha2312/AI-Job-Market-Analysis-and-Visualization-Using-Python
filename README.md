# AI Job Market Analysis and Visualization Using Python

## Description

This project performs exploratory data analysis (EDA) on an AI job market dataset using Python. It uses Pandas for data processing and Matplotlib and Seaborn for visualization to analyze trends in salary, industry demand, experience level, and AI adoption.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Dataset Features

The dataset contains information about:

* Job Role
* Industry
* Country
* Year
* Experience Level
* AI Adoption Level
* Salary
* Education Requirement Level

## Visualizations

The project generates multiple visualizations including:

* Line Chart
* Pie Chart
* Scatter Plot
* Bar Chart
* Histogram
* Box Plot
* Count Plot
* Violin Plot
* Heatmap

## How to Run

1. Install required libraries
   pip install pandas matplotlib seaborn

2. Download the dataset and place it in the project folder.

3. Run the Python script
   python project.py
