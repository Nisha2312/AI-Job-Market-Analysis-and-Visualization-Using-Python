# AI Job Market Data Analysis

## Project Overview

This project analyzes an AI job market dataset using Python. The goal is to explore trends in salary, experience level, industry demand, and AI adoption levels. Data visualization techniques are used to understand patterns and insights from the dataset.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Dataset

The dataset used in this project contains information such as:

* Job Role
* Industry
* Country
* Year
* Experience Level
* AI Adoption Level
* Salary
* Education Requirement Level

## Features of the Project

* Data loading and preprocessing using Pandas
* Data visualization using Matplotlib and Seaborn
* Analysis of salary distribution
* Comparison of salary by experience level
* Visualization of AI adoption trends

## Graphs Generated

The project includes different types of graphs such as:

* Line Chart
* Pie Chart
* Scatter Plot
* Bar Chart
* Histogram
* Box Plot
* Count Plot
* Violin Plot
* Heatmap

## How to Run the Project

1. Install required libraries:
   pip install pandas matplotlib seaborn

2. Download the dataset and place it in the project folder.

3. Run the Python script:
   python project.py

## Project Objective

The objective of this project is to demonstrate how Python can be used for data analysis and visualization to extract useful insights from real-world datasets.
